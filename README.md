# Pocket Monster TCG (口袋怪獸卡牌遊戲)

這是一個極簡、高效且移動優先（Mobile-first）的卡牌收集與對戰遊戲。本專案不需任何開發環境，只要瀏覽器即可遊玩！

## 🌟 遊戲特色

- 🎁 **即時抽卡**: 全螢幕開卡特效，每次獲得 5 張隨機卡片。
- 🗃️ **收藏進度**: 隨時查看你的卡牌庫，並支援多種屬性排序（HP、ATK、SPD）。
- ⚔️ **回合制對戰**: 挑選 10 張卡片組成出戰牌組，挑戰 AI 競技場。
- 📈 **強化機制**: 重複抽到的卡片會自動轉化為爆擊率。

## 🚀 如何開始遊玩

### 直接開啟
1. 下載本專案。
2. 直接使用瀏覽器開啟 `index.html`。

### 部署至 GitHub Pages
如果你想分享給朋友：
1. 將此專案上傳至 GitHub。
2. 進入該 Repo 的 **Settings > Pages**。
3. 在 **Build and deployment** 下方的 Branch 選擇 `main` (或 `master`) 並點擊 **Save**。
4. 稍等幾分鐘，GitHub 就會提供你一個遊戲網址！

## 🗂️ 專案結構

- `index.html`: 遊戲的核心邏輯、介面與樣式。
- `cards.js`: 龐大的卡片資料庫供遊戲讀取。
- `.github/workflows/deploy.yml`: 自動部署腳本，讓你的 GitHub Pages 永保最新。
